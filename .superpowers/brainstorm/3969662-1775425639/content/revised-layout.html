Key Design Decisions

  • Stacked area chart — shows aggregate memory over time, with per-container color breakdown. Time range selector (1h/6h/24h/7d).
  • 85% pressure threshold — dashed red line on chart so you can see when you're approaching the admission-control cutoff.
  • Container table — dense rows with inline memory bars. Color-coded dots match chart legend. Expandable to show forks.
  • Fork sub-table — not cards. PID, state, memory, jobs, current execution, uptime. Scales to dozens of forks per container.
  • No config dialog — removed entirely. No min/max workers UI.
  • Responsive — max-width ~900px centered container.
  • Persisted history — backend stores periodic snapshots for chart data, survives page refreshes.
